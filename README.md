# Zestora

Static marketing site for Zestora — honest, nutrient-dense snacks.

## Local preview

Open `index.html` in a browser, or from this folder run:

```bash
npx --yes serve .
```

Then visit the URL shown (usually http://localhost:3000).

## Deploy on GitHub Pages

1. Push this repository to GitHub.
2. In the repo: **Settings → Pages**.
3. Under **Build and deployment**, set **Source** to **GitHub Actions**.
4. The included workflow publishes the site on every push to `main`.

The live URL will be `https://suhas781.github.io/zestora/` (after the first successful workflow run).
